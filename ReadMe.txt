Тестовое задание:

1. Реализовать REST API на FastAPI.
2. Написать один эндпоинт, который будет принимать GET запросы с параметром "rub_amount" в виде нецелого числа с запятой или точкой.
3. Эндпоинт должен возвращать ответ в формате JSON с ключом "usd_amount", который содержит сумму в долларах.
4. Для получения курса валют использовать API ЦБРФ.
5. Написать рест клиента и рест сервер.
6. Реализовать unit тесты для API.
7. Добавить Swagger для API.
